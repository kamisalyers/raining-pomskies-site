# Raining Pomskies Website Starter

This is a free-to-deploy Next.js website starter for your breeder website.

## What is included
- Midnight-styled homepage
- Available puppies section
- Parent dog section
- FAQ section
- Testimonials and contact section
- Tailwind CSS styling

## Run it locally
1. Install Node.js
2. Open a terminal in this folder
3. Run:

```bash
npm install
npm run dev
```

4. Open `http://localhost:3000`

## Put it on GitHub
1. Create a new GitHub repository
2. Upload all files from this folder
3. Commit the files

## Deploy free on Vercel
1. Sign in to Vercel with GitHub
2. Import your GitHub repository
3. Click Deploy
4. Your site will go live on a free `.vercel.app` link

## Replace placeholders
- Swap the empty image blocks with real puppy photos
- Update contact info in `app/page.jsx`
- Change puppy names, prices, dates, and statuses in `app/page.jsx`
